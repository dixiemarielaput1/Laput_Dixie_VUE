# Laput_Dixie_VUE
Welcome to Extreme Weather in Canada 🍁 This interactive app, highlights Canada’s wildest weather events. From blizzards in the Prairies to coastal floods in the Maritimes, this project connects a custom Lumen API to a Vue.js frontend to deliver real-time details on extreme weather—no umbrella required! ☔❄️🔥

## 🛠️ Installation
No installation required! Just open the index file in your Google Browser and you're all set! 🚀

## 🖥️ Usage
Open the index.html file in your Google Browser.
Enter your body measurements, and let the estimator work its magic! 🎩✨
